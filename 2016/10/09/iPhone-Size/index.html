iPhone Size

2016-10-09 | iOS

icon 和 launch image是每个APP必不可少的元素，好看的图标和启动图会更加地吸引用户。随着iPhone屏幕尺寸越来越多，为了适配各种机型，开发者需要为APP配置不同尺寸的图片。系统会根据图片尺寸，辨别适配的是哪个机型。

一、iPhone size

手机型号 屏幕尺寸 英寸
iPhone 4,4s 320 * 480 3.5 英寸
iPhone 5,5s 320 * 568 4 英寸
iPhone 6,6s,7 375 * 667 4.7 英寸
iphone 6 plus,6s plus,7 plus 414 * 736 5.5 英寸

二、App icon

2.1 iPhone Spotlight

  • iOS 5-6
    base: 29pt, 需要 @2x, @3x，尺寸： 58 x 58, 87 x 87
  • iOS 7-10
    base: 40pt, 需要 @2x, @3x，尺寸：80 x 80, 120 x 120

2.2 iPhone App

  • iOS 7-10
    base: 60pt，需要 @2x, @3x，尺寸：120 x 120, 180 x 180

2.3 iPhone Settings

  • iOS 5-10
    base: 29pt，需要 @2x,@3x，尺寸：58 x 58, 87 x 87

2.3 iPhone Notification

  • iOS 7-10
    base: 20pt，需要 @1x,@2x，尺寸：20 x 20, 40 x 40

2.4 iPad Spotlight

  • iOS 7-10
    base: 40pt, 需要 @1x, @2x，尺寸：40 x 40, 80 x 80

2.5 iPad App

  • iOS 7-10
    base: 76pt，需要 @1x, @2x，尺寸：76 x 76, 152 x 152

2.6 iPad Pro App

  • iOS 9-10
    base: 83.5pt，需要 @2x，尺寸：167 x 167

2.7 iPad Settings

  • iOS 5-10
    base: 29pt，需要 @1x,@2x，尺寸：29 x 29, 58 x 58

2.8 iPad Notification

  • iOS 7-10
    base: 20pt，需要 @1x,@2x，尺寸：20 x 20, 40 x 40

2.9 尺寸总结

工程适配iOS8.2以上，需要的图片尺寸：

  1. 29 x 29
  2. 57 x 57
  3. 58 x 58
  4. 80 x 80
  5. 87 x 87
  6. 114 x 114
  7. 120 x 120
  8. 180 x 180

注意：iOS所有图标的圆角效果由系统自动生成，UI给出的图标本身并不需要圆角效果。

三、launch image

启动图、闪屏图、APP应用市场引导图尺寸：

  1. 640 * 960
  2. 640 * 1136
  3. 750 * 1334
  4. 1242 * 2208
