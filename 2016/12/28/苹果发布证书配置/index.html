苹果发布证书配置

2016-12-28 | Document

创建APP对应的Distribution Certification（发布证书 ），Distribution Provisioning Profile( 发布授权文件 )，ID，流程如下：

一、 登录开发者证书中心

官网：https://developer.apple.com/account/ios/certificate

二、创建发布证书

点击右上角+号按钮，进入如下页面。可以看到有两种类型的证书，一种是开发证书（Development），另一种发布证书（Production）。我们需要发布一个APP，所以选择发布证书中的 App Store and Ad Hot。

点击底下的 Continue 进入下个页面

大意就是需要创建一个CSR文件以及所需步骤

三、创建CSR文件

1） 在自己电脑找到 钥匙串访问 （应用程序–>其他），钥匙串访问–>证书助理–>从证书颁发机构请求证书。如图

填写邮箱地址，选中存储到磁盘，点击继续

四、回到刚才 步骤二 的页面，点击 Continue

进入这个页面，点击 Choose File 选中刚才创建的CSR文件，点击 Continue ，这样证书就创建成功了。点击 Download 下载证书，双击安装到电脑。

五、创建ID

证书创建完毕以后，选择 App IDs

点击 + 好按钮，创建ID，Name一般是你APP名字的全拼组成（原则：能标识这个app就好，如果APP名字很长，可以缩减），不能包含特殊字符（@, &, *, ‘, “），Bundle ID 按反域名的格式填写，如：com.chuangjiang.app，为了方便自动化打包配置文件格式如下：

  • 以 com. 开头和以 .app 结尾；
  • 中间是刚才创建的Name，如：com.chuangjiang.app；

底下的 App Services，除了默认勾选的以外，我们只要再勾选一项 Push Notifications 就可以了，然后 Continue，下个页面点击 Register，这样APP ID就创建完成。

六、创建推送证书

选中你刚才创建的APP ID，点击 Edit，

然后点击红色框内的 Create Certificate，又会进入创建CSR页面，我们前面已经创建过了，Continue 就行了。

选择我们前面创建的CSR文件，Continue

推送证书创建成功。

七、创建描述文件

选择 Provisioning Profiles –> All，点击 + 号按钮创建描述文件

选中 App Store ,然后 Continue。进入选择App ID页面，选中你刚才创建的 App ID，然后 Continue。

进入证书选择页面，选中最开始创建的发布证书，然后 Continue；

接下来会让你为这个描述文件起一个名字，格式如下：

  • App ID中的 Bundle Name + Appstore，如：chuangjiangAppstore

最后，点击 Continue。

这样发布一个APP所需要的证书就全部创建完成了。
